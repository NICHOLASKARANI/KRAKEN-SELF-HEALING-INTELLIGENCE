## RECORDING TIPS:

1. **Screen Recording**: Use OBS Studio (free)
2. **Resolution**: 1920x1080
3. **Audio**: Clear microphone, enthusiastic tone
4. **Background**: Clean desktop, only KRAKEN visible
5. **Actions**: Click buttons slowly, explain each step

## POST-PRODUCTION:

1. Add background music (low volume)
2. Add text overlays at key moments
3. Speed up loading screens
4. Add zoom effects on important metrics
